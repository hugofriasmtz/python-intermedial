## Checklist rápido

- [ ] Crear un DataFrame desde una lista de diccionarios.
- [ ] Crear un DataFrame desde un diccionario de listas.
- [ ] Leer un CSV sin ayuda.
- [ ] Revisar `shape`, `columns`, `index` y `dtypes`.
- [ ] Seleccionar columnas y filas básicas.
- [ ] Resumir una columna numérica.
- [ ] Filtrar filas con condiciones.
- [ ] Detectar nulos por columna.

> [!IMPORTANT]
> Si vas a modificar columnas originalmente leídas desde CSV, guarda una copia si necesitas preservar los datos originales.
> Documentación completa: [pandas docs](https://pandas.pydata.org/docs/)
