# Bloque 2 - Retos de Matplotlib

En este bloque practicaras personalizaciones y graficos un poco mas avanzados.

## Objetivo

- Aprender a comunicar mejor una historia con los datos.
- Practicar elementos visuales como subplots, anotaciones y escalas.
- Mantener los graficos claros y faciles de leer.

## Retos incluidos

1. `reto_1_subplots.py` - Comparar dos conjuntos de datos en subplots.
2. `reto_2_anotaciones.py` - Agregar anotaciones a una tendencia.
3. `reto_3_escala_logaritmica.py` - Usar escala logaritmica cuando los datos crecen mucho.
4. `reto_4_barras_comparativas.py` - Comparar dos grupos con barras agrupadas.

## Reglas de practica

- Lee primero el enunciado completo.
- Resuelve sin copiar soluciones externas.
- Intenta que el grafico explique algo claro.
- Usa titulos, ejes y leyenda cuando haga falta.
